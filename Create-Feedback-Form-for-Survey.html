Lab: Create Feedback Form for Survey

Create a feedback form using React functional components and manage user details using the useState hook. Implement event handlers, validate inputs, handle form submissions, and display a confirmation dialog using the confirm method before final submission.

Learning objectives

  • Create a form to collect user feedback, including their name, email, and the feedback message
  • Handle form submission with data validation to ensure users enter their name and provide feedback
  • Display a confirmation dialog showing the entered information before final submission
  • Reset the form fields to provide a clean form for submitting additional feedback

Prerequisites

  • Basic knowledge of HTML
  • Intermediate knowledge of JavaScript
  • Basic knowledge of React function component, useState hook, and handling forms

5 Implement Change Handlers

1

Define a handleChange function after the useState initialization:

jsx 
const handleChange = (event) => {
  const { name, value } = event.target;
  setFormData({
    ...formData,
    [name]: value
  });
};
const handleChange = (event) => { ... }
Arrow function that receives the event triggered by user interaction with an input element.
const { name, value } = event.target
Extracts the name attribute and current value from the input field that triggered the event.
setFormData({ ...formData, [name]: value })
Updates state immutably — spreads existing formData and overwrites only the changed field. A new object is created rather than modifying state directly.

7 Handle Form Submission

1

Add the handleSubmit function just before the return of the FeedbackForm component:

jsx 
const handleSubmit = (event) => {
  event.preventDefault();

  const confirmationMessage = `
  Name: ${formData.name}
  Email: ${formData.email}
  Feedback: ${formData.feedback}
  `;

  const isConfirmed = window.confirm(
    `Please confirm your details:\n\n${confirmationMessage}`
  );

  if (isConfirmed) {
    console.log('Submitting feedback:', formData);
    setFormData({
      name: '',
      email: '',
      feedback: ''
    });
    alert('Thank you for your valuable feedback!');
  }
};
event.preventDefault()
Prevents the default browser form submission behavior (page reload).
confirmationMessage
A template literal that constructs a readable summary of the user's inputs.
window.confirm(...)
Displays a browser dialog. Returns true if user clicks OK, false if they click Cancel.
setFormData({ ... })
Resets all form fields to empty strings after a successful submission.
alert(...)
Displays a thank you message to the user after form reset.

Full solution — FeedbackForm.jsx 
import React, { useState } from 'react';
import './FeedbackForm.css';

const FeedbackForm = () => {
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    feedback: ''
  });

  const handleChange = (event) => {
    const { name, value } = event.target;
    setFormData({ ...formData, [name]: value });
  };

  const handleSubmit = (event) => {
    event.preventDefault();
    const confirmationMessage = `
    Name: ${formData.name}
    Email: ${formData.email}
    Feedback: ${formData.feedback}
    `;
    const isConfirmed = window.confirm(
      `Please confirm your details:\n\n${confirmationMessage}`
    );
    if (isConfirmed) {
      console.log('Submitting feedback:', formData);
      setFormData({ name: '', email: '', feedback: '' });
      alert('Thank you for your valuable feedback!');
    }
  };

  return (
    <>
      <nav><h1>Tell Us What You Think</h1></nav>
      <form onSubmit={handleSubmit} className="feedback-form">
        <h2>We'd Love to Hear From You!</h2>
        <p>Please share your feedback with us.</p>
        <input type="text" name="name" placeholder="Your Name"
          value={formData.name} onChange={handleChange} />
        <input type="email" name="email" placeholder="Your Email"
          value={formData.email} onChange={handleChange} />
        <textarea name="feedback" placeholder="Your Feedback"
          value={formData.feedback} onChange={handleChange}></textarea>
        <button type="submit">Submit Feedback</button>
      </form>
    </>
  );
};

export default FeedbackForm;

Extend the form with a 1–5 star rating system using radio buttons.

Add a rating field to the formData state, initialized as an empty string:

  feedback: '',

2

Add five radio buttons inside the form (values 1–5), each using the same handleChange handler:

jsx 
<p>Rate Us:</p>
{[1, 2, 3, 4, 5].map((val) => (
  <label key={val}>
    <input
      type="radio"
      name="rating"
      value={val}
      onChange={handleChange}
    />
    {val}
  </label>
))}

The handleChange function requires no changes — the spread operator already handles the new rating field automatically.

3

Update handleSubmit to include the rating in the confirmation message:

jsx 
const rating = formData.rating;
const confirmationMessage = `
Name: ${formData.name}
Email: ${formData.email}
Feedback: ${formData.feedback}
Rating: ${rating}

Also reset rating in setFormData when clearing the form after submission:

jsx 
setFormData({ name: '', email: '', feedback: '', rating: '' });
5

Re-run the app and submit the form. The confirm dialog now shows the rating alongside name, email, and feedback.

Full solution — FeedbackForm.jsx with Rating 
import React, { useState } from 'react';
import './FeedbackForm.css';

const FeedbackForm = () => {
  const [formData, setFormData] = useState({
    name: '', email: '', feedback: '', rating: ''
  });

  const handleChange = (event) => {
    const { name, value } = event.target;
    setFormData({ ...formData, [name]: value });
  };

  const handleSubmit = (event) => {
    event.preventDefault();
    const rating = formData.rating;
    const confirmationMessage = `
    Name: ${formData.name}
    Email: ${formData.email}
    Feedback: ${formData.feedback}
    Rating: ${rating}
    `;
    const isConfirmed = window.confirm(
      `Please confirm your details:\n\n${confirmationMessage}`
    );
    if (isConfirmed) {
      console.log('Submitting feedback:', formData);
      setFormData({ name: '', email: '', feedback: '', rating: '' });
      alert('Thank you for your valuable feedback!');
    }
  };

  return (
    <>
      <nav><h1>Tell Us What You Think</h1></nav>
      <form onSubmit={handleSubmit} className="feedback-form">
        <h2>We'd Love to Hear From You!</h2>
        <p>Please share your feedback with us.</p>
        <input type="text" name="name" placeholder="Your Name"
          value={formData.name} onChange={handleChange} />
        <input type="email" name="email" placeholder="Your Email"
          value={formData.email} onChange={handleChange} />
        <textarea name="feedback" placeholder="Your Feedback"
          value={formData.feedback} onChange={handleChange}></textarea>
        <p>Rate Us:</p>
        {[1, 2, 3, 4, 5].map((val) => (
          <label key={val}>
            <input type="radio" name="rating" value={val}
              onChange={handleChange} />{val}
          </label>
        ))}
        <button type="submit">Submit Feedback</button>
      </form>
    </>
  );
};

export default FeedbackForm;
